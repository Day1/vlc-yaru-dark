# VLC Yaru Dark

A VLC media player Skin for Ubuntu 20.04 with [Yaru theme](https://github.com/ubuntu/yaru). The skin is based on [GNOME-style client-side decorations (CSD) initiative](https://wiki.gnome.org/Initiatives/CSD).
